# Tischreservierung — Бронирование столиков

Учебный фронтенд-проект: одностраничное приложение для бронирования столиков в ресторане. Реализовано на чистом HTML/CSS и React без сборщика — React, ReactDOM и Babel подключены через CDN, JSX компилируется прямо в браузере.

**Демо:** https://movi2752.github.io/Frontcourse/

## Возможности

- Регистрация и авторизация пользователя (данные хранятся в `localStorage`)
- Главная страница со списком ресторанов
- Интерактивная схема зала с выбором свободного столика
- Создание, просмотр и отмена бронирований
- Личный кабинет с историей бронирований
- Адаптивная вёрстка

## Стек

- HTML5, CSS3
- React 18 (через UMD-сборку с unpkg)
- Babel Standalone — для трансформации JSX в браузере
- `localStorage` в роли хранилища данных


## Как запустить локально

Поскольку `index.html` подгружает `.jsx`-файлы как внешние скрипты, открыть его двойным кликом из файловой системы не получится — браузер заблокирует запросы по схеме `file://`. Нужен любой локальный HTTP-сервер.

Например, через Python:

```bash
python -m http.server 8000
```

Или через Node.js:

```bash
npx serve .
```

После этого открыть `http://localhost:8000` в браузере.

## Деплой

Проект развёрнут на GitHub Pages из ветки `main`. Никакой сборки и CI не требуется — статика отдаётся как есть.

## Заметки

Использование Babel Standalone в браузере подходит только для учебных и демо-целей: на каждой загрузке страницы JSX компилируется заново, что заметно медленнее обычной сборки через Vite или Webpack. Для продакшена этот подход не годится.
